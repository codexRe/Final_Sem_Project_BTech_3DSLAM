# Autonomous Driving Prototype
This prototype is based on Turtlebot3 using Jetson nano with Ubuntu 20.04 image.

## Getting started with robot setup:
* Download the <a href="https://ln5.sync.com/dl/741c98fe0/x8kxkhgs-cgmzk7rf-n4m7pyw8-h64tzbv5">Ubuntu20.04</a> image.
* Directly use <a href="https://www.balena.io/etcher">balena etcher</a> (recommended) to burn the image on jetson board.
* Start the jetson board. (password: jetson)
* Run the following in terminal:\
<code>chmod 755 ./robot.sh</code>\
<code>sudo ./robot.sh</code>
